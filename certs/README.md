# Certifications

All the certification study hubs live here, each a self-contained folder built to the same
standards (Mermaid diagrams, source-grounded, no fabrication, a `Sources` section per page).
Shared fundamentals they all draw on — [foundations/](../foundations/README.md),
[prerequisites/](../prerequisites/README.md), [protocols/](../protocols/README.md),
[reference/](../reference/README.md), [learning/](../learning/README.md) — stay at the repo root.

| Hub | What it is | Focus |
|-----|-----------|-------|
| 🛡️ **[wallix/](wallix/README.md)** | WALLIX / PAM — the repo's primary track | WCA → WCP → WCE, Bastion & suite, OT/IDaaS/IAG |
| ⚔️ **[ceh/](ceh/README.md)** | EC-Council Certified Ethical Hacker v13 | Offensive (educational/defensive framing) |
| 🧱 **[security-plus/](security-plus/README.md)** | CompTIA Security+ (SY0-701) | Vendor-neutral foundational baseline |
| 🔵 **[cysa-plus/](cysa-plus/README.md)** | CompTIA CySA+ (CS0-003) | Blue-team / SOC analyst — detection & response |
| 🟠 **[pentest-plus/](pentest-plus/README.md)** | CompTIA PenTest+ (PT0-003) | Vendor-neutral penetration testing + reporting |
| 🔴 **[oscp/](oscp/README.md)** | OffSec OSCP / OSCP+ (PEN-200) | Hands-on offensive (24-hour exam) |
| 🟣 **[pnpt/](pnpt/README.md)** | TCM Security PNPT | Practical engagement + live debrief |
| 🧩 **[adjacent-certs/](adjacent-certs/README.md)** | One-page overviews | CISSP, cloud security (AZ-500 / AWS) |
