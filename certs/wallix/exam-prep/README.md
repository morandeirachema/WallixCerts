# Exam preparation

Everything to get ready for the WALLIX certification exams (each is a final
multiple-choice questionnaire requiring **70% to pass** — see the
[certification framework](../overview/certification-framework.md)).

| Page | Use it to |
|------|-----------|
| [Study plan](study-plan.md) | Follow an ordered path through this repo for your target cert (WCA-P, WCP-P, WCE-P, or a track), with suggested time estimates |
| [Practice questions](practice-questions.md) | Self-test with 54 unofficial MCQs by topic, each with an explained answer linking back to the source doc |
| [Cheat sheet](cheat-sheet.md) | Cram the dense facts: object model, ports, auth methods, password defaults, HA facts |

> ⚠️ The practice questions are **unofficial study aids written for this repo — not real
> WALLIX exam questions.** They are grounded in the documented, sourced facts.
