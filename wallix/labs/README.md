# Labs — hands-on practice

Learning PAM sticks when you build and break things. These pages cover how to get a
practice environment and what to do in it.

| Page | What it gives you |
|------|-------------------|
| [Building a home lab](building-a-home-lab.md) | The official WALLIX Academy lab topology & hardware specs, where Bastion can be obtained, hypervisor options, and what you can practice on a self-built AD + Windows + Linux substrate |
| [Hands-on exercises](hands-on-exercises.md) | Seven guided practice walkthroughs (first authorization, password check-out, approval workflow, recording/audit, external auth, HA replication, REST API) mapped to the certification labs |

> ⚠️ The official WALLIX Academy labs are provided **only to enrolled trainees**. Where
> access is gated, these pages say so rather than pointing at unverified downloads.

➡️ Each exercise links to the [deep dive](../deep-dives/README.md) that explains it.
